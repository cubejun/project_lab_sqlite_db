# 프로젝트랩 3조 발표자료


# 1. 목적
* 센스햇의 센서를 이용해 측정한 데이터를 sqlite3 DB에 저장
* DB에 저장된 데이터 웹 페이지에 실시간 그래프 구현
* LED 매트릭스와 조이스틱을 이용한 미니게임
